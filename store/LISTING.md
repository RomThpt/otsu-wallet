# Otsu Wallet -- Store Listing

## Short Description (132 chars max)

```
A secure, open-source XRPL wallet extension with EVM sidechain support, token management, and cross-chain bridging.
```

## Chrome Web Store -- Detailed Description

```
Otsu Wallet is a security-first browser extension for the XRP Ledger and XRPL EVM Sidechain.

KEY FEATURES

- Create and manage multiple XRPL accounts from a single wallet
- Full EVM Sidechain support with native XRP and ERC-20 tokens
- Cross-chain bridge between XRPL and EVM Sidechain via Axelar
- Send and receive XRP, issued tokens, and ERC-20 tokens
- View transaction history with detailed breakdowns
- Real-time DEX price quotes (XRP/USD)
- Trustline management for XRPL tokens
- Token list with metadata and icons
- Escrow and check tracking
- NFT display with IPFS metadata resolution
- Reserve breakdown with USD valuations

SECURITY

- All private keys encrypted locally with AES-GCM and PBKDF2 (600K iterations)
- No data ever leaves your browser -- zero external servers
- Manifest V3 with minimal permissions
- Granular dApp permission scopes (read, sign, submit, network)
- Open-source under MIT License -- verify the code yourself

WALLET IMPORT

- 24-word mnemonic phrase
- XRPL secret key (s...)
- Family seed
- Private key (hex)
- Xumm secret numbers

SUPPORTED NETWORKS

- XRPL Mainnet and Testnet
- XRPL EVM Sidechain Mainnet and Testnet

FOR DEVELOPERS

- Standard XRPL provider API for dApp integration
- EIP-1193 compatible provider for EVM dApps
- Permission-scoped access control

Otsu Wallet is free, open-source software. No accounts, no subscriptions, no data collection.

Source code: https://github.com/RomThpt/otsu-wallet
Privacy policy: https://github.com/RomThpt/otsu-wallet/blob/main/PRIVACY.md
```

## Firefox AMO -- Detailed Description

```
Otsu Wallet is a security-first browser extension for the XRP Ledger (XRPL) and XRPL EVM Sidechain.

Features:
- Create and manage multiple XRPL accounts
- EVM Sidechain support (native XRP + ERC-20 tokens)
- Cross-chain bridge via Axelar
- Send/receive XRP, issued tokens, and ERC-20s
- Transaction history with details
- Real-time DEX pricing
- Trustline and token management
- Escrow and check tracking
- NFT display with IPFS resolution

Security:
- Local-only encryption (AES-GCM + PBKDF2 600K iterations)
- No external servers, no analytics, no tracking
- Manifest V3, minimal permissions
- Granular dApp permission scopes
- Open-source (MIT License)

Import formats: mnemonic, secret key, family seed, hex private key, Xumm numbers.

Networks: XRPL Mainnet/Testnet, EVM Sidechain Mainnet/Testnet.

Source: https://github.com/RomThpt/otsu-wallet
```
