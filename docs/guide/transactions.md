# Transactions

Otsu supports sending and receiving assets on both XRPL and EVM networks.

## Sending XRP

1. Click **Send** on the home screen
2. Enter the destination address
3. Enter the amount
4. Optionally add a **destination tag** (required by some exchanges)
5. Optionally add a **memo**
6. Review and confirm the transaction

## Sending Tokens

1. Select the token from the **Tokens** tab
2. Click **Send**
3. Enter the destination and amount
4. Review and confirm

## Transaction History

The **Activity** tab shows your full transaction history including:

- Payments sent and received
- Trustline changes
- NFT operations
- Contract interactions (EVM)

Each transaction shows the type, amount, counterparty, and status.

## Cross-Chain Bridge

Otsu integrates with the Axelar bridge for transferring assets between XRPL and the EVM Sidechain:

1. Open the **Bridge** section
2. Select the source and destination chains
3. Enter the amount to bridge
4. Review fees and confirm
5. Monitor the bridge status in the activity feed
