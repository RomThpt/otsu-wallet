# Tokens & NFTs

Otsu supports viewing and managing tokens and NFTs across XRPL and EVM chains.

## XRP & Token Balances

Your main balance (XRP or the native EVM token) is displayed on the home screen. Additional tokens appear under the **Tokens** tab.

### Trustlines (XRPL)

On XRPL, holding a token requires setting up a trustline to the token's issuer. Otsu allows you to:

- View existing trustlines and balances
- Set up new trustlines
- Remove trustlines (if balance is zero)

### ERC-20 Tokens (EVM)

On the EVM Sidechain, ERC-20 tokens are displayed automatically when detected. You can also manually add tokens by contract address.

## NFTs

### XRPL NFTs

Otsu fetches and displays your XRPL NFTs (XLS-20). You can view metadata, images, and transfer NFTs.

### Viewing NFTs

Navigate to the **NFTs** tab to see your collection. Click on any NFT to view its details including:

- Token ID
- Issuer
- URI and metadata
- Transfer fee
